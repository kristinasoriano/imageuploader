Running this script locally, you need to do the following:

1. Download the zip file

2. Extract the file and copy the folder

3. Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Run your local server, then open PHPMyAdmin in your web browser(http://localhost/phpmyadmin)

5. Create a database with name ImageUploader

6. Import imagetable.sql file(given inside the zip package in SQL file folder)

7. Run the script http://localhost/ImageUploader
